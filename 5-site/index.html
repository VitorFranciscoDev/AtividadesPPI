O que e a Web?

A Web e um sistema de paginas conectadas pela internet.

Voce acessa sites usando um navegador como Chrome ou Firefox.

Os sites ficam guardados em servidores e quando voce digita um endereco o navegador busca a pagina.

A Web usa protocolos como HTTP e HTTPS para funcionar.

O que e HTML?

HTML significa HyperText Markup Language. E a linguagem usada para criar paginas web.

Voce usa tags como <div>, <p>, <h1> para estruturar o conteudo.

HTML define o que aparece na pagina: textos, imagens, links, botoes.

Todo site tem HTML, e o esqueleto da pagina.

O que e CSS?

CSS significa Cascading Style Sheets. E usado para deixar as paginas bonitas.

Com CSS voce muda cores, tamanhos, fontes, espacamentos.

Voce pode usar classes e ids para selecionar elementos e aplicar estilos.

CSS deixa o site com cara profissional e organizado.

O que e Animate?

Animate serve para fazer coisas se moverem na tela. Voce usa @keyframes para criar a animacao.

Exemplo: animation: mover 2s;

Voce pode fazer coisas girarem, sumirem, mudarem de cor, etc.

As principais propriedades sao: animation-name, animation-duration, animation-delay.

O que e Position?

Position serve para mudar onde um elemento fica na pagina.

Tem varios tipos: static (normal), relative (relativo), absolute (absoluto), fixed (fixo).

Quando voce usa position, pode usar top, left, right, bottom para mover o elemento.

Fixed deixa o elemento sempre no mesmo lugar mesmo quando voce rola a pagina.

O que e Flex?

Flex e uma forma facil de organizar coisas na tela.

Voce coloca display: flex; no elemento pai e os filhos ficam organizados.

Pode usar justify-content para alinhar na horizontal.

E align-items para alinhar na vertical.

E muito usado para fazer menus e organizar conteudo.

O que e Grid?

Grid e tipo uma tabela para organizar elementos.

Voce define quantas colunas e linhas quer com grid-template-columns e grid-template-rows.

Exemplo: grid-template-columns: 1fr 2fr 1fr; cria 3 colunas.

Grid e bom para fazer layouts mais complexos tipo sites inteiros.

Use gap para dar espaco entre os elementos.
