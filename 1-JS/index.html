O que é JavaScript?

JavaScript foi criado para tornar páginas web interativas, permitindo que o navegador reagisse a ações do usuário sem precisar recarregar tudo pelo servidor. Quando o código é executado no navegador, ele roda em um ambiente restrito chamado sandbox, que limita o acesso ao sistema do usuário por segurança. Já no servidor, como no Node.js, o JavaScript pode acessar arquivos, rede e outros recursos do sistema operacional. No navegador, objetos globais como document, console e alert são fornecidos pelo próprio ambiente: document controla o HTML da página, console serve para registrar mensagens de depuração e alert mostra caixas de aviso ao usuário.
